# Oro-plata-indicadores-ciclos-económicos
Este proyecto analiza si los precios del oro y la plata contienen información predictiva sobre la actividad económica futura en Estados Unidos.

Se utilizan datos mensuales (2012–2022) y un modelo de regresión lineal dinámico con errores robustos Newey-West para evaluar el efecto de las variaciones interanuales rezagadas de ambos metales sobre:

Producción Industrial

Tasa de Desempleo

## Resultados principales

El oro se asocia con menor producción industrial futura y mayor desempleo.

La plata se asocia con menor desempleo futuro, sin efecto significativo sobre la producción industrial.

Los resultados se mantienen controlando por volatilidad y COVID.

## Conclusión

Los metales preciosos, especialmente el oro, parecen incorporar información relevante sobre la evolución futura del ciclo económico, aunque los resultados no deben interpretarse como causales.
